## Installation

```bash
- Create a .env file using .env.sample as a reference/template

$ npm install
```

## Migrations

```bash
# genearate
$ npm run migration:generate customName

# create
$ npm run migration:create customName

# run
$ npm run migrations:run

# revert
$ npm run migrations:revert
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
